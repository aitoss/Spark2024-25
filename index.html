FAQs

01

Do I need prior programming knowledge to attend SPARK?

Not at all! SPARK is designed to start from the basics and gradually build up to more advanced topics. It is open to all FEs, regardless of their branch or previous coding experience. Whether you’re a beginner or have some coding background, you’re very welcome to join! SPARK will run from 10 August to 29 September, with sessions held every weekend. Each session will be approximately 2 hours.

02

What topics will be covered in the bootcamp?

The bootcamp will cover the following topics:

1. Introduction to Web Development

2. HTML & CSS Basics

3. JavaScript Fundamentals

4. Responsive Web Design

5. Version Control with Git & GitHub

03

Will there be any assignments or projects?

Yes, participants will be given small practice assignments regularly. There will also be a two competitions - REPLICA and FSoC (FE Season of Code), during the run of SPARK. Here participants can apply everything they’ve learned to build fully functional websites and learn how to contribute to open source.

04

What resources will I need?

You will need a laptop with an internet connection. We also recommend installing the code editor - Visual Studio Code, but detailed instructions on setting up your development environment will be provided in the first session. Don't forget to bring your own USB cables and chargers. All reference materials and notes will be shared on the community and website.

05

What if I’m already familiar with web development?

Absolutely! While the bootcamp starts with the basics, it quickly progresses to more advanced topics. Experienced participants can also assist others and potentially take on more challenging projects. The competitions organised are exciting opportunities to win prizes and goodies.

06

Who do I contact if I have more questions?

For any additional questions, feel free to contact the OSS Club via email at aitoss@gmail.com
